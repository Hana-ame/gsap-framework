# Examples

Example components demonstrating every feature of the Hana-ame UI framework: games, UI demos, media playback, visual effects, architecture patterns, and legacy compatibility.

## Categories

- **Games** — minesweeper, snake, breakout, tetris, conway, 2048, colony, life-map
- **UI Demos** — window, multi-window, single-window, window-canvas, confirm, loading, scrollable, text-input, ui-helpers, layers, registry
- **Media** — image, clickable-image, scrollable-image, video-player, video-player-dom, audio-viz, drawing, filters, starfield, particle-rain, picture-drag, clock
- **Effects** — typing-effect, gsap, waves, cutscene, cutscene-minimal, tutorial variants
- **Architecture** — backend-controlled, fullscreen, infinite
- **Legacy** — single, multiple, window, window-mobile, screen-size, pixi-confirm
